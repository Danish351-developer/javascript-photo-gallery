About This Website

This website was created as my final project for the Coursera CSS course. It demonstrates the use of semantic HTML, accessible navigation, CSS Grid, Flexbox, hover effects, and responsive web design techniques.

I enjoy learning web development and improving my skills in HTML and CSS. Creating this project helped me understand how to build accessible, attractive, and well-structured web pages.
